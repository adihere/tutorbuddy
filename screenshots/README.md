# Screenshots for TutorBuddy Documentation

This directory contains screenshots referenced in the main README.md file. 

## Required Screenshots

1. **landing-page.png** - The landing page with key connection button and feature highlights
2. **tutor-form.png** - The input form where users enter topic, select subject, age, and output mode
3. **processing.png** - The loading screen while AI generates content
4. **tutorial-view.png** - The main lesson display with formatted content and audio controls
5. **quiz-view.png** - The interactive quiz with questions and feedback
6. **image-gallery.png** - The auto-rotating slideshow of educational visuals
7. **parent-report.png** - The guardian insights report with performance metrics

## Image Guidelines

- Recommended size: 1920x1080 pixels (16:9 aspect ratio)
- Format: PNG or JPEG
- Should show the full browser window with visible UI elements
- Ensure any sensitive information (API keys, personal data) is obscured
- Use consistent browser window size across all screenshots for consistency

## Adding New Screenshots

When updating screenshots:
1. Take new screenshots following the guidelines above
2. Replace the existing files in this directory
3. Update the main README.md if adding new screenshot references
4. Ensure all screenshots reflect the current UI and functionality
